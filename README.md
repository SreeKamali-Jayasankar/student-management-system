## Student Management System

## Overview

The  Student Management System is a Python-based mini project that allows users to manage student records through a simple menu-driven interface. It demonstrates the use of Python fundamentals such as functions, lists, dictionaries, file handling, exception handling, and JSON data storage.

## Features

* Add a new student
* Update student details
* Delete a student
* Search for a student by ID
* Display all students
* Save student records to a JSON file
* Load student records automatically when the program starts
* Validate Student ID input (numeric IDs only)
* Prevent duplicate Student IDs

## Technologies Used

* Python 3
* JSON (for data storage)
* Visual Studio Code

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/sreekamali-jayasankar/Student-Management-System.git
```

2. Navigate to the project folder:

```bash
cd Student-Management-System
```

3. Run the program:

```bash
python main.py
```

## Sample Menu

```text
===== Student Management System =====

1. Add Student

```

## Data Storage

Student records are stored in a `students.json` file. Any changes made through the application are automatically saved and reloaded the next time the program starts.
